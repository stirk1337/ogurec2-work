Ogurec Bot
===========

Требования
----------
- Python 3.14+
- uv или pip для управления зависимостями
- Токен Discord бота и ключ Tenor API

Установка
---------
Настройка переменных среды (файл `.env` в корне)
```
DISCORD_BOT_TOKEN=xxx
TENOR_API_KEY=xxx
PREFIX=!
BOT_CHAT_ID=749662464538443948
MAIN_CHAT_ID=670981415306788870
```

Основные возможности
--------------------
- Ответы на упоминания и случайные сообщения/реакции в чате.
- Стикеры/эмодзи по триггерам, счётчик сообщений.
- Периодическое обновление статуса активности с постом в основной канал и GIF.
- Кастомная роль “Ребрендинг” с напоминаниями и очередью.
- Slash-команда `/hello`.

Команды и события
-----------------
- Slash: `/hello` — приветствие пользователя.
- Авто: реакции/ответы на сообщения (вероятностно).
- Авто: напоминание по ребрендингу (раз в 168 часов).
- Авто: обновление presence (каждый час) + GIF пост в основной канал по триггеру.
